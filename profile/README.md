## Propose a project

If you'd like to propose a community project to be hosted in this organization, please reach out to `emma[@]blueprint[.]zip`. This can be a project you maintain, or a project that should be maintained by the community.

Please note that:

- We will not host (install) scripts or websites.
- Repository READMEs will have a "community project" disclosure at the beginning, that should not be removed.
- Blueprint community projects must provide value to the community.
- AI-generated content is prohibited.
